# alexa-skills
This repository contains miscellaneous skills written for Amazon Alexa, mostly jokes for friends in my home. As such, it is unlikely that they will every be published en masse as public skills. Feel free to download and use the skills. Don't forget to enter your own app id into the code.
Skills include such gems as (mostly) rude responses to queries such as "Alexa, tell me I'm pretty"

Some of the wording used in these skils could be considered inappropriate or offensive. Browse at your own discretion.
